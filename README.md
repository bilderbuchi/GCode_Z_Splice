# GCode_Z_Splice

`GCode_Z_Splice` is a program which splices Gcode files at given Z coordinates, 
to enable users to combine e.g. fine and coarse versions of a mesh for Fused 
Deposition Modeling.

Created using Python, wxPython, and wxFormBuilder
